Exercise: Hotel Bookings - Data Wrangling

Download the Hotels & Data Wrangling R-markdown file from Blackboard and open it in R-Studio.

Instructions:

  1. Set yourself as the “author” of the document.

  2. Solve the problems in Exercises 1 - 8.

  3. After you make the changes to the markdown document, save your work. Make sure that the document has no errors and can be “knit” successfully.

  4. Save and rename the file as “yourname_hotels.Rmd”.

  5. Upload your revised markdown document to the assignment in Blackboard.

Lab: Nobel Laurates

Download the zip folder containing the Nobel Lab markdown document and supporting files, and open it in R-Studio.

Instructions:

  1. Knit the markdown document to make sure the file compiles successfully.
  2. Edit the markdown document as instructed.
  3. Rename and upload the completed markdown file to the assignment in Blackboard.
