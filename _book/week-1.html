Exercise: UN Votes

Watch this video that introduces the AE-01 UN Votes assignment (8 minutes).

Video

Note: You will not complete the assignment in R Studio Cloud as indicated in the video, but will instead complete the activity in your RStudio program on your computer. Download the UN Votes R-Markdown document from Blackboard, and open it in RStudio. When you make changes to the markdown document, and then knit it, the changes will be saved.

Instructions:

  1. Insert your name as the “author”.

  2. (Part 1) In the paragraph below the Introduction, respond to these questions for the countries France, United Kingdom, and United States.

  • How do various countries vote in the United Nations General Assembly?
  • How have their voting patterns evolved throughout time?
  • How similarly or differently do they view certain issues?
  1. (Part 2) Change the graph to show the data for three other countries.
  • Modify the code to make this change.
  • Save all changes.

  1. Rename the markdown file with your name. For example if your name is Joe Smith, name the file “jsmithunvotes.Rmd”, and save the file to your computer.

  2. Upload your revised markdown document to the assignment in Blackboard.

Exercise: Bechdel

Download the Bechdel R-markdown file from Blackboard. Open it in RStudio.

Instructions:

  1. Insert your name as the “author”.

  2. Update the markdown document as instructed in the markdown file.

  3. Save and rename the file as “yourname_bechdel.Rmd”.

  4. Upload your revised markdown document to the assignment in Blackboard.
